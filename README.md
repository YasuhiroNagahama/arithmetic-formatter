# Name

**Arithmetic Formatter**

# Explain

入力された計算問題を縦書きにして表示します。ファイル実行時に、いくつかの注意が書かれているので、それに従って使用して下さい。
<br><br>例

**入力**

235 \+ 52

**出力**

&nbsp;&nbsp;235  
\+&nbsp;&nbsp;52  
\-----  
&nbsp;&nbsp;287

# Features

* オブジェクト指向プログラミングを行いました。
* 間違った形式でコマンドが入力された場合に、エラー文が出るようにしました。（すべてのエラーをとりきれるかはわからない）
